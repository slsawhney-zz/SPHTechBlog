INTRODUCTION
------------

SPHTechBlog intends to Create Author and Blog content type. They are binded with each other through Entity reference.

REQUIREMENTS
------------

No special requirements.

INSTALLATION
------------

 * Extract Zip data to DRUPAL_ROOT/modules/custom
 * The Extract directory structure would be similar to 
       DRUPAL_ROOT/modules/custom/sphtechblog/author
       DRUPAL_ROOT/modules/custom/sphtechblog/blog
 * Go to DOMAIN/admin/modules Or Admin -> Extend ->List
 * Find 'SPHTECHBLOG'
 * Install both Author and Blog module

CONFIGURATION
-------------

 * No Additional configuration is required

CONTENT ADDITION
----------------

 * Go to DOMAIN/node/add/author Or Admin -> Content -> Add Content -> Author to add authors
 * Go to DOMAIN/node/add/blog Or Admin -> Content -> Add Content -> Blog to add blog

LIST
----------------

 * To List the added content go to DOMAIN/blog
